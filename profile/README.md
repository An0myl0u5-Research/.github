## Mission

- Make research changes reviewable and attributable.
- Favor deterministic, rebuildable outputs over runtime widgets.
- Publish operational status clearly: implemented, policy-gated, or planned.

## Public roadmap (near-term)

1. Publish a public ARC index for governance, contribution paths, and visibility into private/public boundaries.
2. Publish dashboard artifacts (JSON/Markdown/charts) with scheduled rebuilds and commit history.
3. Add measurable integrity metrics (citation coverage, release hygiene, signing posture).

## Contribution model

Contributors are welcome from technical and non-technical backgrounds. Workstreams and templates will be published in the ARC index as they become public.
